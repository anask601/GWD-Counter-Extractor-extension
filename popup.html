🔍 How to Use

1Open GWD Page

Navigate to your Google Web Designer project page (preview.html or index.html)

2Copy Index URL (Optional)

If on preview.html, click "📋 Copy Index URL" to get the index.html URL, then navigate manually

3Extract Counters

Click "🔍 Extract Counters" to scan the page for gwd-counter elements

4Copy Results

Use "📋 Copy" to copy all extracted counter elements to your clipboard

💡 Tips

  • ⚠️ Yellow warnings indicate NaN values that may cause tracking issues
  • 🔗 Copy Index URL converts preview.html to index.html automatically
  • 📱 Extension works on any localhost port or published GWD page
  • 🔍 Extract works on both preview.html and index.html pages
